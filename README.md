# QA Automation Project (Selenium + TestNG)

## 🔧 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports

## 📂 Features
- Login Automation
- Add to Cart
- Checkout Flow
- Page Object Model (POM)
- Screenshot on Failure
- HTML Report Generation

## ▶️ How to Run
1. Clone the repository
2. Open project in Eclipse
3. Run testng.xml

## 📊 Reports
- Generated using Extent Reports
- Available in /reports folder

## 📸 Screenshots
- Captured automatically on test failure
- Available in /screenshots folder

## 🎯 Scenario Covered
- User login
- Product selection
- Add to cart
- Checkout process
